
# Dioxus Tutorial

## Usage

```bash
# Create project
dx new NAME
# 1. Choose Template: Bare-Bones
# 2. Choose Fullstack: false
# 3. Choose Router: false
# 4. Choose Tailwindcss: false
# 5. Chooose Prompts for LLMs: false
# 6. Choose Platform Default: Desktop

# Run project
cd NAME && dx serve
# NOTE: DON'T use `cargo run`

# Check project
cd NAME && cargo check
```
